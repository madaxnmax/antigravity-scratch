# Email Analyzer with AI Flask App

Follow the steps below to set up the Flask app locally on a Windows operating system.

## Setup Instructions

### Step 1: Clone the repository

- Clone the repository using the link below

    ```bash
    git clone https://github.com/EganLegacyPartners/AiModel.git
    ```

- After cloning the repository, navigate to the cloned directory

    ```bash
    cd AiModel
    ```

### Step 2: Create a virtual environment using Python 3

```bash
python3 -m venv env_app
```

### Step 3: Activate the virtual environment

- Use the following command to activate the virtual environment on Windows

    ```bash
    .\env_app\Scripts\activate
    ```

### Step 4: Install dependencies from requirements.txt

```bash
pip install -r requirements.txt
```

### Step 5: Run the Flask app

```bash
python app.py
```

### Step 6: Access the app in your web browser

- Open your browser and go to: http://localhost:5000
- The app will be accessible on port 5000 by default
